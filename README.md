# DeckBuilding Game
Unity로 제작한 덱빌딩 기반의 카드게임

## 기능
- 카드 전투
- 턴제 시스템
- 적군
- 플레이어와 적 HP 표시
- 공격 카드 버튼
- 강공격 카드 버튼
- 힐 카드 버튼
- 적 반격 시스템
- 승리/패배 결과 표시
- 덱에서 랜덤으로 드로우
- 덱/손패/묘지 표시
-  ScriptableObject 생성


## 사용기술
- Unity
- C#

## 오류목록
- Random 함수 유니티엔진, 시스템함수 충돌 오류 해결
- ScripttableObject로 변경 후 데이터타입 오류
