# MyCardGame
Unity로 제작한 덱빌딩 기반의 카드 전투게임

# 프로젝트 소개
플레이어는 다양한 카드를 사용하여 적을 처치하며 스테이지를 진행
전투를 이기면 카드 보상을 획득하고 덱에 추가하여 보스를 처치하는 것이 목표.
ScriptObject를 활용하여 데이터 관리
CardInstance 구조를 통해 카드 개별 강화 시스템을 구현

## 주요 기능
# 전투 시스템
- 턴제 전투
- 플레이어 HP / 적 HP
- 적 반격
- 승리/패배 판정
- 전투 로그 출력
- 최근 5개 로그 표시

# 카드 시스템
- 공격 카드
- 강공격 카드
- 회복 카드
- 방어 카드
- 연속 공격 카드
- 공격 + 회복 카드
- 공격 + 자해 카드
- 카드 설명 생성

# 신규 카드
- 연속 공격 (5 데미지 x 2회)
- 흡혈 공격 (공격 + 회복)
- 광전사 (공격 + 자해)
- 철벽 (방어)
- 폭발 (공격 + 자해)

# 덱 시스템
- 덱
- 손패
- 묘지
- 셔플
- 카드 강화
- 카드 제거
- 전체 카드 수 표시

# 적 시스템
- 슬라임
- 고블린
- 오크
- 연속 전투
- 보스

# 보스 패턴
- 보스 여부 판정
- 특수 공격 데미지 설정
- 특수 공격 주기 설정
- 일반 / 보스 행동 분리
- 특수 공격 랜덤 데미지

# 방어 시스템
- 방어도 획득
- 피해 감소

## 보상 시스템
# 카드 보상
- 적 처치 후 보상 카드 3장 표시
- 카드 1장 선택
- 선택 카드 덱 추가
- 보상 선택 중 카드 사용 제한

# 카드 강화
- 강화 가능한 3장 표시
- 플레이어 선택 강화
- 이미 강화된 카드 제외
- 강화 시 이름에 + 표시
- 강화 수치 증가

# 카드 제거
- 제거 가능한 카드 3장 표시
- 플레이어 선택 제거
- 제거 후 카드 수 갱신
- 제거 로그 출력

## 이벤트 시스템

## 데이터 구조
# CardData (ScriptableObject)
카드 원본 데이터 관리
- 카드 이름
- 카드 타입
- 데미지
- 회복량
- 방어도
- 강화 수치
- 연속 공격 여부
- 공격 횟수
- 자해 데미지

# CardInstance
실제 덱에 존재하는 카드 1장 관리
- 강화 여부 저장
- 강화 적용 후 수치 계산
- 카드 설명 생성
- 카드 희귀도 표시
적용 내용
- CardData 원본 보호
- 카드별 개별 강화 상태 저장
- 동일 카드 개별 강화 지원

# EnemyData (ScriptableObject)
- 적 이름
- 최대 체력
- 공격력
- 보스 여부
- 특수 공격 데미지
- 특수 공격 주기
- 적 행동 리스트

# EnemyActionDta (ScriptableObject)
- 행동 이름
- 행동 타입
- 데미지
- 연속 공격 횟수
- 방어도 증가량
- 방어도 무시 수치
- 행동 확률

# 코드 개선 사항
- DrawCard/DisCardHand 역할 분리
- 보상 선택, 스테이지 진행 2번에 걸쳐 드로우를 해서
  스테이지 진행 시에만 드로우로 변경
- CardData / CardInstance 구조 분리
- 카드 개별 강화 시스템 적용
- 카드 설명 생성 로직 개선
- 이벤트 발생 여부 함수 분리
