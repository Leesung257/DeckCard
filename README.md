# MyCardGame
Unity로 제작한 덱빌딩 기반의 카드 전투게임

# 프로젝트 소개
플레이어는 다양한 카드를 사용하여 적을 처치하며 스테이지를 진행
전투를 이기면 카드 보상을 획득하고 덱에 추가하여 보스를 처치하는 것이 목표.
ScriptObject를 활용하여 데이터 관리

## 기능
전투 시스템
- 턴제 전투
- 플레이어 HP
- 적 HP
- 적 반격
- 승리/패배 판정

카드 시스템
- 공격 카드
- 강공격 카드
- 회복 카드
- 방어 카드

덱 시스템
- 덱
- 손
- 묘지
- 셔플

적 시스템
- 슬라임
- 고블린
- 오크
- 연속 전투

방어 시스템
- 방어도 획득
- 피해 감소

보상 시스템
- 적 처치후 보상 3장 표시
- 보상 카드 1장 선택
- 선택한 카드 묘지에 추가
- 선택 중 전투 카드 사용 불가

스테이지 진행 시스템
- 보상 획득 후 다음 스테이지 진행 표시
- 선택 시 다음 몬스터 등장

코드 리팩토링
- DrawCard/DisCardHand 역할 분리
- 보상 선택, 스테이지 진행 2번에 걸쳐 드로우를 해서
  스테이지 진행 시에만 드로우로 변경

보스 패턴 시스템
- EnemyData 기반 보스 여부
- 보스 특수 공격 데미지 설정
- 특수 공격 데미지 주기 설정
- 노말 적과 보스 행동 분리
- 특수 공격 데미지 랜덤성

UI 시스템
- 전체 카드 수 표시
- 덱 카드 수 표시
- 손패 수 표시
- 묘지 수 표시
- 스테이지 표시
- 적 공격력 표시

## 클래스 구조
BattleManager
- 전투진행
- 카드 사용
- 적 관리
- 보상 처리
- 스테이지 진행

CardData (ScriptableObject)
- 카드 이름
- 카드 타입
- 데미지
- 회복량
- 방어도

EnemyData (ScriptableObject
- 적 이름
- 최대 체력
- 공격력
