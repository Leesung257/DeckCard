# My project
Unity로 제작한 덱빌딩 기반의 카드게임

## 기능
- 카드 전투
- 턴제 시스템
- 적군
- 플레이어와 적 HP 표시
- 공격 카드 버튼
- 강공격 카드 버튼
- 힐 카드 버튼
- 적 반격 시스템
- 승리/패배 결과 표시
- 덱에서 랜덤으로 드로우
- 덱/손패/묘지 표시

## 시스템
- 공격 카드 : 적에게 10데미지
- 강공격 카드 : 적에게 20데미지
- 힐 카드 : 플레이어 HP 5 회복
- 적 HP가 0이 되면 승리
- 플레이어 HP 0이 되면 패배
- 덱에서 랜덤으로 드로우
- 드로우/버리기/덱이 없으면 다시 채우기
- 일정 숫자만큼 적 등장

## 사용기술
- Unity
- C#
